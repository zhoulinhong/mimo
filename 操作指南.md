# 个人主页操作指南

## 修改内容

用 VSCode 打开 `index.html`，修改以下内容后 `Ctrl+S` 保存：

| 位置 | 说明 |
|------|------|
| `<h1 class="hero-name">张三</h1>` | 姓名 |
| `const roles = [...]` | 职位标签 |
| `<p class="hero-tagline">` | 一句话介绍 |
| `<p class="about-text">` | 关于我段落 |
| `<div class="stat-number">` | 数据卡片（年限/项目数等） |
| 联系方式卡片 | GitHub、邮箱、微信等 |

## 预览

双击 `index.html` 用浏览器打开，修改后按 `F5` 刷新。

## 部署到 GitHub Pages

### 首次部署（只需做一次）

```bash
# 1. 在 github.com/new 创建仓库（Public，不勾选 README）

# 2. 本地推送代码
git init
git add index.html
git commit -m "初始化个人主页"
git remote add origin https://github.com/你的用户名/仓库名.git
git branch -M main
git push -u origin main
```

### 开启 Pages

1. 打开 `github.com/你的用户名/仓库名/settings/pages`
2. Branch 选 `main`，点 **Save**
3. 等 1-2 分钟即可访问

### 访问地址

```
https://你的GitHub用户名.github.io/仓库名/
```

### 后续更新内容

修改 `index.html` 后：

```bash
git add index.html
git commit -m "更新内容"
git push
```

Pages 会自动更新，无需任何额外操作。

## Git 身份配置（仅首次需要）

```bash
git config --global user.email "你的邮箱"
git config --global user.name "你的GitHub用户名"
```
